# 8-Week-SQL-Challenge
Danny Ma의 [8-Week SQL Challenge](https://8weeksqlchallenge.com/)에 8가지 케이스에 대한 저의 솔루션을 작성하고 있습니다.

## 목차
- [Case Study #1 - Danny’s Dinner](https://github.com/waterymallon/8-Week-SQL-Challenge/tree/main/Case%20Study%20%231%20-%20Danny's%20Diner)
- [Case Study #2 - Pizza Runner](https://github.com/waterymallon/8-Week-SQL-Challenge/tree/main/Case%20Study%20%232%20-%20Pizza%20Runner)

## 개발 환경

* **Database**: MySQL Community Server 8.0.44
* **Language**: SQL
* **Tools**: MySQL Workbench 8.0.46

MySQL Enterprise Edition을 설치할 필요없이 Community로도 충분합니다.

설치링크: [MySQL Community Downloads](https://dev.mysql.com/downloads/)

- MySQL Installer for Windows에서 Setup Type을 Developer Default로 다운받으셔도 되고,<br>
Custom으로 Server와 Workbench만 다운받아도 상관없습니다.
- 단 Installer for Windows를 통해 Workbench를 다운받을 때 중간에 멈추는 오류가 발생하는 경우가 흔합니다. (Microsoft Visual C++ Redistributable 패키지 설치 충돌 때문),
이 경우 Workbench Installer로 개별다운 받는 방법이 있습니다.
