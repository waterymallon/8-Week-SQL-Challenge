## Tech Stack

* **Database**: MySQL Community Server 8.0.44
* **Language**: SQL
* **Tools**: MySQL Workbench 8.0.46

The **MySQL Community Edition** is sufficient for this challenge; there is no need to install the Enterprise Edition.

**Download Link**: [MySQL Community Downloads](https://dev.mysql.com/downloads/)

* You can select the **"Developer Default"** setup type in the MySQL Installer for Windows, or choose **"Custom"** to install only the Server and Workbench.
* **Note**: The Windows Installer often hangs or freezes when downloading/installing Workbench (usually due to conflicts with the Microsoft Visual C++ Redistributable package). If this happens, I recommend downloading and installing the **Workbench Installer separately**.
